Mars Rover kata
===============

Prerequisites
-------------

Install [NodeJS with NPM](http://nodejs.org/).

Run following commands:

```bash
npm install -g grunt-cli
npm install -g bower
npm install -D gulp-jasmine
npm install
bower install
```
